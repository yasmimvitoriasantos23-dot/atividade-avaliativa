# 🎬 Supernatural Fan Page

Este é um projeto web simples e interativo dedicado à série **Supernatural (Sobrenatural)**. O projeto consiste em uma página institucional/fan page que apresenta informações sobre a série (como sinopse e número de temporadas) e inclui recursos interativos de engajamento e acessibilidade.

## ⚙️ Funcionalidades

- [x] **Exibição de Conteúdo:** Apresentação do título, sinopse, temporadas e imagem promocional dos irmãos Winchester.
- [x] **Contador de Curtidas Dinâmico:** Um botão interativo que incrementa o número de curtidas em tempo real ao ser clicado.
- [x] **Acessibilidade (Zoom de Texto):** Botões dinâmicos (`A+` e `A-`) que aumentam ou diminuem o tamanho da fonte de toda a página em blocos de `2px`, melhorando a leitura.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web Front-End:

- **HTML5:** Estruturação e semântica do conteúdo.
- **CSS3:** Estilização visual e layout (através do arquivo `style.css`).
- **JavaScript (ES6):** Manipulação da DOM para criar a interatividade do contador e dos botões de acessibilidade.
